Como crear un Objeto

 

1)      Seleccionar el proyecto en el cual se desea trabajar.

 


 

2)      Seleccionar OBJETOS y luego Botón Crear un OBJETO.

 


 

3)      Completar las propiedades generales (ladrillo)

 


 

a.      Clase: define el tipo de Objeto

·         ABMS:  Formulario

·         CUADRO:  Browse

·         FILTRO

·         GRAFICO

·         HOJA DE DATOS

·         LISTA

b.      Fuente de Datos:  indicar la base de datos.

c.      Nombre:  Permite dar a más de uno el mismo nombre.

d.      Descripción

 

4)      Completar las propiedades particulares (lapicito)
