Manejo de creacion de consultas_php
------------------------------------
		
	Se ingresa con: archivo, clase, metodo

	-> Existe archivo?
		* NO
			-> Crear archivo
			-> Crear clase
			-> Crear metodo
		* SI
			-> Existe clase?
				* NO
					-> Hay codigo?
						* NO
							-> Crear clase
							-> Crear metodo
						* SI *******************> ERROR! Se pide cambiar el archivo.
				* SI
					-> Existe metodo?
						* NO
							-> Crear metodo al final
						* SI
							*******************> Se avisa que se va a sobreescribir un metodo
							-> Reemplazar metodo
