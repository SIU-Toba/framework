toba_vinculador

Resúmen de la Clase | Métodos | Propiedades
Esta clase maneja la VINCULACION entre ITEMS. Conoce todos los lugares a los que el

ITEM actual puede acceder (considerando el USUARIO que lo solicito) Para navegar hacia estos items puede construir URLs e incluirlos en algún TAG

Crea un vinculo hacia un item 


			Parámetros:

			

					

				string  
				
				$proyecto: 
				
				 Proyecto destino, por defecto el actual			

					

				string  
				
				$item: 
				
				 Item destino, por defecto el actual			

					

				array  
				
				$parametros: 
				
				 Parametros pasados al item, es un arreglo asociativo id_parametro => valor			

					

				array  
				
				$opciones: 
				
				 Arreglo asociativo de opciones ellas son:  
  1.  zona => Activa la propagación automática del editable en la zona
    2. 

  2.  cronometrar => Indica si la solicitud generada por este vinculo debe cronometrarse
    3. 

  3.  param_html => Parametros para la construccion de HTML. Si esta presente se genera HTML en vez de una URL. Las claves asociativas son: frame, clase_css, texto, tipo [normal,popup], inicializacion, imagen_recurso_origen, imagen
    4. 

  4.  texto => Texto del vínculo
    5. 

  5.  menu => El vinculo esta solicitado por una opción menu?
    6. 

  6.  celda_memoria => Namespace de memoria a utilizar, por defecto el actual
    7. 

  7.  servicio => Servicio solicitado, por defecto get_html
    8. 

  8.  objetos_destino => array(array(proyecto, id_objeto), ...) Objetos destino del vinculo
    9. 

  9.  prefijo => Punto de acceso a llamar.
    10. 

			

				


	

	

  

    
    	Return:  Una URL o el link html en caso
