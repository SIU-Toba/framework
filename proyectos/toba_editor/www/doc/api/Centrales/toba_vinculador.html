				array

				 Arreglo asociativo de opciones ellas son:                      zona => Activa la propagación automática del editable en la zona,                      cronometrar => Indica si la solicitud generada por este vinculo debe cronometrarse,                      param_html => Parametros para la construccion de HTML.
