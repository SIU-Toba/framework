ef_checkbox


Resúmen de la Clase | Métodos Heredados | Métodos | Propiedades

Clase base de los elementos de formulario. El checkbox tiene un valor que depende si esta chequeao o no, por eso set_estado no afecta al check sino sólo a su value. Para cambiar el check usar chequear(boolean)
