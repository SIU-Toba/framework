class notificacion


Clase estatica para el manejo de mensajería al usuario Su funcionamiento se basa en encolar mensajes con notificacion.agregar() y luego mostrarlos con notificacion.mostrar() Los proyectos pueden variar la forma en que se muestran las notificaciones definiendo un nuevo metodo notificacion.tipo_ventana.
Por defecto notificacion.tipo_ventana = notificacion.ventana_modal
