Desarrollando con Toba

Una vez instalado el framework recomendamos bajar y darle una mirada al Manual PDF porque centraliza toda la documentación que se detalla a continuación:

  • Para el día a día de trabajo:
    • Documentación wiki de referencia, que contiene el detalle de los servicios disponibles.
    • Documentación API php.
    • Documentación API javascript.
    • Una lista de Preguntas frecuentes (FAQ)
  • Hay disponible un demo online de una aplicación creada por toba. Esta demo (llamado proyecto toba_referencia) está incluida en la instalación del framework y puede ser de suma utilidad para investigar conceptos puntuales (cómo utilizar tal o cual componente, cómo hacer llamadas AJAX, etc.)
  • Hay disponible un tutorial online que muestra una introducción a la herramienta
