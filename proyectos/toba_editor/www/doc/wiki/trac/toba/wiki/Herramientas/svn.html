Herramientas/svn

Scripts svn caseros

  • Borrar todos los archivos no versionados bajo el directorio actual: 
    for a in `svn st | grep ? | awk '{ print $2 }'`
do
    rm -R $a
done
  • Sincronizar el directorio actual, borrando los que ya no estan y agregando los archivos nuevos: 
    for a in `svn status | grep "!" | awk '{print $2}'`; do svn remove $a; done
for a in `svn status | grep "?" | awk '{print $2}'`; do svn add $a; done
  • Integrar con Kompare en KDE (archivo config de subversion) 
    kompare "$6" "$7"
