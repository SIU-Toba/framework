Instalacion/ObtenerCodigoFuenteToba

Descargar Fuentes SVN

Toba utiliza  SVN como sistema de control de versiones, la descarga vía este medio es la recomendada ya que permite obtener los últimos arreglos de bug y parches seguridad de cada versión de forma sencilla posteriormente a la instalación.

La url de descarga es http://repositorio.siu.edu.ar/svn/toba/versiones/1.5.0

También es posible acceder mediante cifrado SSL ( https://...) o un acceso alternativo via desarrollos2.siu.edu.ar

Para windows puede utilizar alguno de estos clientes SVN:

  •  TortoiseSVN. Una vez instalado es necesario  obtener una copia de trabajo indicando la dirección http previa.
  •  Cliente línea de comandos. Descargar utilizando el comando svn checkout
