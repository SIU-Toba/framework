PHP GTK

Instalación GNU/Linux

  • Instalar los paquetes necesario para compilar la extensión
    • Si es Debian o derivado, ejecutar 
      sudo apt-get install build-essential gawk automake1.9 autoconf libglib2.0-0 libglib2.0-dev libgtk2.0-0 libgtk2.0-dev libglade2-dev php5-cli php5-dev
    • Si es OpenSUSE ejecutar como root 
      yast -i glib2 glib2-devel gtk2-devel gtk2 libglade2 libglade2-devel php5-zip php5-devel
  • Bajar la ultima versión de http://gtk.php.net/download.php 
    wget http://gtk.php.net/do_download.php?download_file=php-gtk-2.0.0beta.tar.gz
  • Descomprimir 
    tar xfv php-gtk-2.0.0beta.tar.gz
cd php-gtk-2.0.0beta/
  • Compilar e instalar: 
    ./buildconf
./configure
make 
sudo make install
  • Configurar PHP 
    sudo -i
echo extension=php_gtk2.so >> /etc/php5/cli/php.ini
  • Comprobar instalacion con 
    php -m | grep gtk
