Instalación WSO2

Ubuntu

  1. Descargar y descomprimir el tar.gz desde  http://wso2.org/downloads/wsf/php
  2. Instalar paquetes: 
    sudo apt-get install php5-dev libxml2-dev
  3. Compilar el framework 
    ./configure
make
sudo make install
  4. Editar archivo /etc/apache2/php.ini y agregar 
    extension = wsf.so
  5. Reiniciar apache
