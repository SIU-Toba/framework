Proyecto Doc - Referencia

Descripción

Este proyecto surge como extensión de los ejemplos que se ven en la documentación de referencia.

Instalación

Una vez instalado toba:

  1. Instalar un cliente SVN (bajo Windows recomendado TortoiseSVN).
  2. Hacer un checkout del proyecto en el directorio toba/proyectos/referencia:
    1. Por línea de comando: 
      svn co https://toba.siu.edu.ar/toba/proyectos/trunk/referencia toba/proyectos/referencia
    2. Usando un cliente gráfico apuntar a https://toba.siu.edu.ar/toba/proyectos/trunk/referencia
  3. Desde la consola ejecutar: 
    proyecto cargar referencia
  4. Para poder utilizar los ítems que trabajan con una base de datos de ejemplo se debe ejecutar: 
    psql -U usuario -h localhost base_de_referencia $toba_dir/proyectos/referencia/sql/referencia.sql

Se asume que el usuario toba está presente en la instancia.

Ejecución básica

  • Desde el Administrador? seleccionar el proyecto Doc - Referencia.
  • Seleccionar la vista de ítems.
