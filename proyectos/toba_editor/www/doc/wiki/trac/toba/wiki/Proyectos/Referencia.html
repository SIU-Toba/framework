Proyecto Referencia

Este proyecto surge como extensión de los ejemplos que se ven en la documentación de referencia. Existe una versión online del proyecto, aunque para entender en profundidad las operaciones es recomendado instalarlo localmente.

versión online navegable

Instalación Local

Si no se utilizó la instalación automática del framework, es necesario instalar manualmente este proyecto de ejemplo.

1- Cargar en el proyecto en la instancia actual 

 toba proyecto cargar -p toba_referencia

2- Instalar el proyecto 

 toba proyecto instalar -p toba_referencia
