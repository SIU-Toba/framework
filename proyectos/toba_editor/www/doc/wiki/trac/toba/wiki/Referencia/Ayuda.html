Referencia/Ayuda

Ayuda Contextual

En los distintos elementos del proyecto definidos en el editor se permite ingresar una descripción asociada, formando una ayuda contextual del elemento en cuestión.

Niveles en que se muestra

Operación

En la operación se muestra un ícono de ayuda al lado del nombre de la operación, al pasar sobre el icono se le muestra (sin retardo) la descripción asociada.

Componente

En todos los componentes que salgan a interface se puede incluir una descripción en forma de tooltip que aparece al costado del título del componente.

Otra opción es forzar a que se muestre como una barra propia, sin tooltip con un ícono de información. Esto se puede hacer configurando al componente con el método set_modo_descripcion(false)

Campo o columna

Cuando en un formulario se tienen Campos o efs es posible a cada uno de estos asociarles una descripción en forma de tooltip . Lo mismo sucede con la ayuda de las utilerias e iconos del ei_arbol

Formato

El formato en general de la ayuda es HTML, aunque se puede incluir referencias al wiki. Por ejemplo 

Usar los [wiki:Referencia/Objetos Componentes] de <strong>persistencia</strong>.

Se mostrará como

Usar los Componentes de persistencia.

También se pueden incluir links a contenido estatico del proyecto: 

Para más info ver el [link:ayuda/manual.pdf Manual].

El link apuntará al directorio navegable del proyecto.

Para URLs absolutas se puede usar: 

Para buscar usar [url:www.google.com Google].
