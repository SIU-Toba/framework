Empezar a trabajar con el toba

- toba instalacion crear - toba instalacion agregar_db -d 'mi_base'

Comenzar a desarrollar un proyecto en una instancia vacia

- toba instancia crear -i 'mi_instancia'

Crear un proyecto

- toba proyecto crear -i 'mi_instancia' -p 'mi_proyecto'

Adjuntar un proyecto existente a una instancia - toba proyecto cargar -p 'mi_proyecto'

Elinar un proyecto

- toba proyecto eliminar -p 'mi_proyecto'

Refrescar el estado de un proyecto - toba proyecto regenerar -p 'mi_proyecto'
