Lista de Cambios

  • Arreglos de Bugs de la 1.0
  • Se implementa el ef_cuit basado en el de Oscar
  • Documentación del API tanto PHP como JS
  • Ventana generar_layout en formulario y pantalla, para cambiar el layout de una manera radical.
  • Generación de subclases acordes con la nueva versión
  • Previsualización de forms y cuadros en el editor
  • Mejoras al login generico, cuando se pide un item explicito en la url inicial y luego se inicia la sesion se redirecciona al item que se pidio originalmente (permite por ejemplo hacer bookmarks o pasar urls por otros medios.
  • Arreglo al menu-css para ie7, tiene problemas con el solapado de niveles, se opta por correr los niveles a la derecha asi no se solapa (menos accesible pero anda)
  • inicializacion.php pasa a ser el metodo "iniciar_contexto" de la sesion.
  • Nuevo proyecto toba_instancia pensado para administrar instancias.
