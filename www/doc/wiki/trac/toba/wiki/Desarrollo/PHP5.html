Inicio / Desarrollo / PHP5.1

Uitilizar PHP 5.1

Ventajas

  • La SPL agrega funcionalidad interesante (Por ejemplo, se puede tener mas de un autoload() ).
  • Se incluye una version supuestamente estable de la PDO (PHP Data Objetcs), que puede llegar a ser interesante para algun experimento en un branch.
  • Aparentemente la performance mejora.

Problema

Algunas librerias que usa el toba tiran un notice: Only variable references should be returned by reference . La nueva version de ADOdb no tiene este problema, pero si simpletest y tal vez alguna otra (ezpdf?, jpgraph110?). Aparentemente lo que dispara ese error tambien lo metieron en la version 4.4.0 de PHP, por lo que seguramente las librerias se van a adecuar en la brevedad.
