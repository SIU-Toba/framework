Inicio / Guia

Funcionalidad estándar

  • Catálogo de clases estándares de TOBA
  • Catálogo de bases de datos conectadas al entorno. Incluye una ventana para ejecutar sentencias SQL
  • Crear nuevas clases genéricas o extender clases existentes
  • Resumen de métodos / funciones contenidas en una clase
  • Creación de nuevos proyectos dentro del entorno

Proyectos

  • Catálogo de operaciones del proyecto
  • Catálogo de objetos del proyecto
  • Catálogo de objetos estáticos?
  • Catálogo de objetos clasificados por la capa arquitectónica en la cual están contenidos. Ejemplo: Elementos de Interface, Controladores de Negocio, etc.
  • Visualizador del código PHP asociado a una clase o archivo PHP
  • Editor de propiedades de los objetos (para todos los objetos creados en forma declarativa)
  • Instanciador de objetos para comprobar su funcionamiento
  • Vista que relaciona operaciones y objetos que la implementan
  • Administración de usuarios de cada proyecto
  • Configurar el estilo gráfico de los distintos componentes del proyecto
  • Niveles de seguridad para acceso a los objetos de cada proyecto

Entorno colaborativo

  • Notas entre desarrolladores
  • Descripción de objetos
  • Mecanismo para bajar un proyecto entero, transferirlo a otro lugar y restaurarlo
