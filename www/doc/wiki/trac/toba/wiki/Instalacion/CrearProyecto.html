Creación de un Proyecto

Supongamos que queremos crear un Proyecto mi_proyecto en una Instancia toba_instancia. Se deben seguir los siguientes pasos:

  1. Configurar la instancia en el archivo \toba\php\instancias.php 
    <?php
    $instancia["toba_instancia"][apex_db_motor] = "postgres7";
    $instancia["toba_instancia"][apex_db_profile] = "168.83.60.71";
    $instancia["toba_instancia"][apex_db_usuario] = "dba";
    $instancia["toba_instancia"][apex_db_clave] = "*dba-";
    $instancia["toba_instancia"][apex_db_base] = "toba_instancia";
?>

  1. Abrir consola (Cygwin)
  2. Comprobar la instancia actual: 
    echo $toba_instancia[enter]
  3. Si la instancia no es toba_instancia entonces setear la variable de la siguiente manera: 
    toba_instancia=toba_instancia[enter]
  4. Crear la instancia toba_instancia 
    instancia iniciar[enter]
  5. Crear el proyecto (este paso crea la carpeta del proyecto dentro de la carpeta proyectos) 
    proyecto crear mi_proyecto[enter]
  6. Crear un usuario para el proyecto (puede crearse de 2 maneras) 
    db run datos_instancia_basicos.sql[enter]

o accediendo a un cliente de postgres (PgAdminIII) En la base toba_instancia en las tablas:

  • Apex_usuario -> crear un usuario
  • Apex_usuario_proyecto -> crear usuario x proyecto
