Requisitos previos

Se deberán tener instalado los siguientes productos antes de comenzar la instalación de SIU-TOBA :

  • Apache.
  • PHP5.
  • PostgreSQL.
  • En el caso de ser instalado en Windows debe instalarse junto con cygwin.
  • Fuentes de Toba.

Apache

  • URL: http://httpd.apache.org/download.cgi.
  • Versión: Es preferible la versión 1.x por recomendación de PHP. En caso de usar 2.x ver precauciones.

PHP

  • URL: http://www.php.net/downloads.php.
  • Versión: 5.x.
  • Extensiones mínimas necesarias:
    • php_pgsql

PostgreSQL

  • URL: http://www.postgresql.org/download/.
  • Versión: 7.4 o mayor.
  • Recomendable instalar un cliente como PgAdminIII.

Cygwin

  • URL: http://www.cygwin.com/.

Fuentes de Toba

Toba utiliza SVN como sistema de control de versiones, para adquirir la última versión de desarrollo del sistema:

  1. Instalar un cliente SVN (bajo Windows recomendado TortoiseSVN).
  2. Hacer un checkout del proyecto:
    1. Por línea de comando: 
      svn co https://toba.siu.edu.ar/toba/trunk toba

Bajo linux, en caso de tener problemas con el encoding, se prefiere utilizar iso88591 tanto para el entorno como default charset para Apache. Ver más acerca del encoding.

    1. Usando un cliente gráfico apuntar a https://toba.siu.edu.ar/toba/trunk
  2. Agregar el path directorio_toba/php al include_path de php.ini
  3. Copiar el archivo directorio_toba/toba.conf a directorio_apache/conf/toba.conf
  4. Incluir el archivo toba.conf en el archivo directorio_apache/conf/httpd.conf: 
    include "conf/toba.conf"
  5. Modificar el punto de acceso toba de toba.conf para apuntar al lugar de instalación.
  6. Reiniciar Apache.
