Testing Automático en PHP

Una de las metologías recomendadas? es la construcción de testing unitario asociado a los distintos módulos del sistema.

Librería utilizada

Actualmente se está utilizando SimpleTest, un clon de JUnit.

Testing en Toba

Requisitos previos

Para que los test se ejecuten correctamente, el proyecto Toba - Testing debe estar en la instancia.

Corriendo los test

Actualmente sólo es posible ejecutar los test desde la consola con el comando: 

instancia testear [-c Categoría] [-t Caso de test particular]

Sin parámetros ejecuta todos los test disponibles en la instancia, se puede especificar una categoría específica o un test en particular. Los casos y sus categorías se puede ver en source:trunk/acciones/pruebas/testing_automatico/casos_dao.php
